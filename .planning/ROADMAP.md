# GSD Project Roadmap

## Project Overview
GSD (Get Shit Done) - A meta-prompting, context engineering and spec-driven development system for Claude Code.

## Current Milestone: v1.x Maintenance

### Phase 1: Output Language Option
**Status:** In Progress
**Goal:** Add output language configuration to allow users to specify the language for generated prompts.

**Requirements:**
- Add language option to configuration
- Modify prompt generation to include language instruction
- Support explicit "use XX language" directive in prompts

**Deliverables:**
- Config option for output language
- Updated prompt templates with language directive
